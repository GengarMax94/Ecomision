## Descripción

EcoMision es un juego de exploración ecológica desarrollado en C++ utilizando programación orientada a objetos.

El jugador controla un explorador que interactúa con diferentes elementos dentro de una reserva natural, recuperando energía, ayudando animales y limpiando residuos contaminantes.
